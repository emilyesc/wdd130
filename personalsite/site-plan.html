Content

Home page

Genshin Impact is a game where your first combat goal is to create a strong character. Your second combat goal is to combine four characters into a single, synergestic, team. Each character has their own elemental alignment, which can be seen when you view any character. They can be either Pyro (fire), Hydro (water), Cryo (ice), Dendro (plants), Geo (earth), Anemo (wind), Electro (lighting). On their own, these elements don't do very much. However, once you combine characters into teams, these elements will react with each other. For example, combining Cryo and Hydro will cause Freeze.
